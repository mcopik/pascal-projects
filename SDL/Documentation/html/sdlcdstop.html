SDL_CDStop

Name

SDL_CDStop -- Stops a CDROM

Synopsis

uses
  SDL;

function SDL_CDStop( cdrom : PSDL_CD ) : Integer;

Description

Stops play on the given cdrom.

Return Value

Returns 0 on success, or -1 on an error.
