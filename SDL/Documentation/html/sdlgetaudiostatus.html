SDL_GetAudioStatus

Name

SDL_GetAudioStatus -- Get the current audio state

Synopsis

uses
  SDL;

function SDL_GetAudioStatus : TSDL_Audiostatus;

Description

  TSDL_Audiostatus = (
    SDL_AUDIO_STOPPED,
    SDL_AUDIO_PLAYING,
    SDL_AUDIO_PAUSED
    );

Returns either SDL_AUDIO_STOPPED, SDL_AUDIO_PAUSED or SDL_AUDIO_PLAYING depending on the current audio state.
